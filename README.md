# Persona-cards
Persona Cards is an interactive and accessible website developed to present the redesign of persona cards, focusing on elderly users with vision and hearing impairments.
The project combines inclusive design, interactivity, and digital accessibility, providing an experience tailored to different user needs.

# Overview
The website offers an intuitive and accessible exploration of the persona card redesign process.
Visitors can view the new versions of the cards, compare before and after changes, and learn more about the purpose and target audience of the project.

# Features
Personas Page: Shows the redesigned persona cards.Each card provides detailed information about the user represented.
Befor vs After: Page comparing the original design and the redesign of the cards. Includes a detailed list of the changes made to each one.
About: Explains the project's objective and design decisions. Includes a persona card for the target user — representing the audience for whom the redesign was intended.

# Accessibility and Dynamic Features
Text Enlargement: Users can enlarge text up to 200%, ensuring readability for people with low vision.
Text-to-Speech (TTS): By clicking or tapping on any text element, the website reads the content aloud. Users can adjust the volume and speech speed directly in the interface.
‘Back to Top’ Button: Allows you to quickly return to the top of the page with a single click.
‘Back to Personas’ Button: Facilitates navigation between individual letter pages and the main list.

# API Integration
The website uses the GitHub API to: dynamically obtain the textual content of each persona card.
